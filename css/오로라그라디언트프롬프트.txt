html, css 코드로 첨부된 이미지처럼 오로라 메시 그라디언트 배경을 만들어줘

요구사항:
- 생성요소는 .exam으로 해주고
- 브라우저의 viewport 크기와 동일한 크기가 되게 해줘 (100% × 100vh)
- HTML 구조는 완전한 HTML5 문서 구조로 해주고
- 모든 스타일은 <style> 태그로 분리해줘 (인라인 스타일 금지)
- CSS로 메시 그라디언트 효과를 만들어줘
- 애니메이션은 적용하지 말고 static으로 만들어줘
- 색상 구성: cyan + purple + yellow (#e7ca0e) + magenta

- 이미지의 텍스트는 위치만 잡아주면 내가 텍스트 내용은 수정할게
- 텍스트는 선명하게 보여야하니까 그라디언트를 주는 요소와 텍스트를 작성하는  html 요소를 분리해서작성해줘
